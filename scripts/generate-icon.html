📱 How to Use These Icons

  1. Right-click on each icon above and save as PNG:
    • Main Icon → save as icon.png (1024×1024)
    • Android Foreground → save as android-icon-foreground.png (432×432)
  2. Move files to assets/images/
  3. Generate all sizes automatically:
    npx @expo/image-utils resize-icons assets/images/icon.png
  4. For Android adaptive icon:
    • Background is solid color (already set in app.json: #E6F4FE)
    • Or create android-icon-background.png with your preferred design

Alternative: Screenshot Method

If right-click save doesn't work:

  1. Take a screenshot of each icon
  2. Crop to exact dimensions
  3. Save as PNG with the filenames above

Quick Command

After saving the icons, run:

npm start -- --clear

This will clear the cache and load your new icon.
